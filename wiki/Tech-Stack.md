# Tech Stack Explained

Derived from `TECH_STACK_EXPLAINED.md` and README.

- FastAPI + Uvicorn: REST interface
- LangGraph: stateful, human-in-the-loop orchestration
- AWS Bedrock (Claude): AI reasoning; Haiku default
- LangChain AWS: Bedrock client integration
- MCP: Model Context Protocol for Claude Desktop tooling
- Pydantic v2: data models and validation

See also: [MCP Setup](MCP-Setup.md), [LangGraph Workflow](LangGraph-Workflow.md).
