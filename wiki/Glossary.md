# Glossary

- Copilot: AI assistant with human-in-the-loop approvals
- LangGraph: Graph-based orchestration framework with state and interrupts
- MCP: Model Context Protocol, tool interface for AI clients like Claude
- Playbook: Structured remediation procedure with commands, verification, rollback
- Root Cause: Primary cause inferred from metrics and context
- RCA: Root Cause Analysis post-incident report
